# 🚀 Laravel Podman Bootstrap

A modern Laravel bootstrap script using **Podman**, with optional support for:

- PostgreSQL
- Redis
- Node.js
- PrimeVue
- Nginx
- Docker-like workflow without Docker

---

## ✨ Features

- Interactive setup
- Works with Podman (rootless)
- Laravel auto-install
- PostgreSQL ready
- Optional Redis
- Optional frontend stack (Node + PrimeVue)
- Clean Nginx config
- Minimal, reproducible setup

---

## 📦 Requirements

- Podman
- curl
- bash
- sed

---

## 🚀 Usage

```bash
chmod +x bootstrap.sh
./bootstrap.sh
